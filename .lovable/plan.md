
# Анализ Тендеров — B2B SaaS платформа

## Обзор
Веб-приложение для анализа тендерной документации по 44-ФЗ, 223-ФЗ и коммерческим закупкам. Пользователи загружают документы, AI анализирует их по 20+ блокам проверки, результат — интерактивный отчёт с возможностью скачивания в PDF и Excel.

---

## Страница 1: Авторизация и личный кабинет
- Регистрация/вход по email (Supabase Auth)
- Личный кабинет с историей всех анализов
- Список прошлых проверок с датой, названием тендера и общим статусом (✅/⚠️/❌)

## Страница 2: Главная / Дашборд
- Краткая статистика: количество проверок, средний уровень риска
- Кнопка «Новый анализ»
- Последние 5 анализов с быстрым доступом

## Страница 3: Загрузка и анализ документов
- Drag & drop зона для загрузки файлов (PDF, Word, Excel, изображения, ZIP)
- Поддержка множественной загрузки (10+ файлов)
- Выбор типа закупки: 44-ФЗ / 223-ФЗ / Коммерческая
- Название тендера (опционально)
- Прогресс-бар обработки с этапами: «Извлечение текста» → «Анализ документов» → «Формирование отчёта»
- Обработка ZIP-архивов: распаковка на бэкенде и анализ каждого файла внутри

## Страница 4: Результаты анализа
- Сводная оценка тендера: общий риск-рейтинг
- Таблица по 20 блокам проверки с колонками: Блок | Статус (✅/⚠️/❌) | Описание риска | Рекомендация
- Фильтрация по статусу (показать только проблемные)
- Возможность раскрыть каждый блок для детального описания
- Кнопки скачивания: PDF-отчёт и Excel-таблица

## Бэкенд (Lovable Cloud)
- **Supabase Auth** — аутентификация пользователей
- **База данных** — хранение анализов, файлов, результатов проверок
- **Storage** — хранение загруженных документов
- **Edge Function: обработка документов** — извлечение текста из PDF/Word/Excel/изображений, распаковка ZIP
- **Edge Function: AI-анализ** — отправка извлечённого текста в Lovable AI (Gemini) с промптом для проверки по 20 блокам
- **Edge Function: генерация отчётов** — формирование PDF и Excel файлов для скачивания

## Дополнительные рекомендации по блокам анализа
К вашим 20 блокам предлагаю добавить:
- **Антидемпинговые меры** — проверка наличия при снижении цены более 25%
- **Электронная подпись** — требования к ЭЦП
- **Соответствие КТРУ** — проверка кодов каталога товаров/работ/услуг
- **Нацпроекты и преференции** — проверка СМП/СОНКО квот

## Дизайн
- Профессиональный, деловой стиль в светлых тонах
- Цветовая индикация рисков: зелёный (✅), жёлтый (⚠️), красный (❌)
- Русскоязычный интерфейс
